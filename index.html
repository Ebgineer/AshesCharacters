Ashes of Creation

Character Selector

Ashes of Creation features four physical archetypes and four mystical archetypes. Each of these is further defined by a secondary archetype from the same set, yielding a total of 64 possible class combinations.

Perhaps you already know which you plan to try. If you're uncertain, this interactive story will guide you to the class suggested by your preferences.
